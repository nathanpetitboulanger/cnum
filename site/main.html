Tout ce dont vous avez besoin

Une suite complète d'outils pour transformer vos feuilles de calcul en calendriers dynamiques.

Parsing Intelligent

Algorithmes avancés pour interpréter les cellules fusionnées et les codes couleurs des emplois du temps complexes.

Normalisation IA

Mapping automatique des initiales vers les noms complets et nettoyage des titres de cours pour une base de données propre.

Analyses Précises

Génération de rapports détaillés sur les volumes horaires par enseignant, groupe et type de formation.

Sync Multi-plateforme

Export vers iCal compatible Google Calendar, Outlook et Apple. Vos cours vous suivent partout.

Vues Personnalisées

Créez automatiquement des feuilles de calcul individuelles pour chaque intervenant en un clic.

Expérience Développeur

Une CLI robuste construite avec Python 3.12 et uv pour une installation et une exécution ultra-fluides.

Prêt à automatiser ?

Suivez ces étapes simples pour déployer CNUM dans votre environnement.

1. Installation des dépendances

# Assurez-vous d'avoir 'uv' installé
uv sync

2. Configuration des clés

Placez votre `token.json` (Google Service Account) à la racine du projet.

3. Exécution du Workflow

# Lancer le parsing et la génération globale
uv run demo.py

Libérez-vous du temps.
